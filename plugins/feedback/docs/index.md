# Introduction

## What is Feedback Plugin?

Feedback plugin is a valuable addition to backstage which allows project managers to get feedbacks for entites in Backstage Catalog.

It is dedicated to simplifying the process of gathering and managing user feedback for service catalog entities. This plugin seamlessly integrates with the feedback-backend plugin and extends its capabilities by allowing users to create Jira tickets associated with their feedback.

## Key features

1. Global Feedback and Bug Listing

   - Provides a comprehensive view of all feedback and bugs submitted across services and components globally. This global perspective helps teams identify recurring issues or common user experiences.

2. Entity-specific Feedback
   - All entity owner can configure their service so that feedbacks option is available to everyone.
   - Along with feedbacks, users can also submit bugs found in that component.
3. Integrated with Issue Tracker
   - The feedback plugin can be easily configured to open tickets for each feedback that is submitted.
   - Currently supported issue trackers are:
     - Jira
     - Github
     - Gitlab
