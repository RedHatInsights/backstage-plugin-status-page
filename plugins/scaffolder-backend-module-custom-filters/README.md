# @appdev-platform/backstage-plugin-scaffolder-backend-module-custom-actions

The custom-actions backend module for the scaffolder plugin.

_This plugin was created through the Backstage CLI_

This plugin exposes 2 filters for the Backstage Scaffolder Plugin.

| Filter | Description |
|---|---|
| kebabCase | Converts string to kebab case. |
| startsWith | Returns true if the given input string starts with the given substring. Otherwise returns false. |
